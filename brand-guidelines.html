01 — Foundation

Brand Identity

Bluebird Café & Diner is a family-owned institution in Arlington, Washington since 1958. The brand blends heritage warmth with a modern luxury aesthetic — refined but approachable, classic but current.

✦
Refined

🌄
Local
Arlington, WA. Pacific Northwest roots. Farm-fresh ingredients. Community first.

"Good food, good people, good morning — every morning since 1958."

Typography

05 — Foundation

Motion & Easing

All transitions use a single custom ease curve. Scroll-triggered reveals use IntersectionObserver at a 10% threshold. Durations stay between 200ms (micro-interactions) and 900ms (page reveals).

Brand Ease
cubic-bezier(0.25, 0.46, 0.45, 0.94)
Default easing for all transitions. Defined as --ease. Never use linear or browser defaults.
Scroll Reveal
opacity + translateY(40px) → 0 · 0.8s
Elements animate in from below via .reveal class + IntersectionObserver. Left/right variants also available.
Hover Lift
translateY(-2px to -4px) · 0.2–0.3s
Buttons lift 2px on hover; cards lift 4px. Always pair with a subtle box-shadow increase.
Float Loop
translateY(0 ↔ -12px) · 5–6s · infinite
Used for hero dish, about emoji, and decorative elements. Creates a calm, living feel.
Counter Ease
1 - Math.pow(1 - progress, 3) · 2000ms
Cubic ease-out for number counters in the stats section. Fires once on scroll into view.
Parallax Hero
translateY(scrollY × 0.35)
Hero content moves at 35% of scroll speed, creating depth without motion sickness.

13 — Usage

Do & Don't

Do

  • Use CSS tokens (--sky, --deep) — never hardcode hex values
  • Use clamp() for all font sizes and section padding
  • Use Cormorant Garamond for all headings and display text
  • Keep button border-radius at 2px
  • Use section labels (small caps + sky line) above every section title
  • Apply the .reveal class to animate content on scroll
  • Use --ease for every CSS transition
  • Keep gold as a rare, premium accent only
  • Always pair a primary button with an outline or ghost variant
  • Use italic in section titles for a secondary colour variation

Don't

  • Don't use more than 3 typefaces on a single page
  • Don't mix gold and sky-blue as co-equal accents in one component
  • Don't use Cabin Sketch outside of decorative chalkboard-style moments
  • Don't use border-radius above 8px — keep the aesthetic sharp
  • Don't animate more than 2 elements simultaneously on load
  • Don't use the custom cursor on touch/mobile devices
  • Don't put body text on the --deep background without significant lightening
  • Don't stack more than two CTAs side-by-side in the hero
  • Don't use box shadows without rgba(11,37,69,…) for brand consistency
